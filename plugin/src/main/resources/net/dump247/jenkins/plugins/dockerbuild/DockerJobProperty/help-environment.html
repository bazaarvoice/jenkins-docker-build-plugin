Set or clear environment variables in the container.

Set a variable: VAR_NAME=VALUE
Unset a variable: unset VAR_NAME

Quoting the variable value is not supported. Values must be on a single line.

Unsetting a variable has no effect if the variable is not already set.

Blank lines or lines that start with # are ignored.

Environment variables from the jenkins agent can be referenced in the value with ${VARIABLE} or ${env.VARIABLE}. System properties from the jenkins agent can be referenced in the value with ${sys.variable}.

Examples:

# a comment unset CLASSPATH SPECIAL_VAR=the value to use OTHER_VAR=${ENV_VAR}_${sys.property.name}
